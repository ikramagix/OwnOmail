OwnOmail is a minimalist plugin that customizes the sender email and name for all WordPress-generated emails. It’s designed to be straightforward, efficient, and does exactly what it supposed to—plain and simple.

== Description ==
- Change your WordPress sender name and address with a few clicks.
- Keeps your WordPress installation clean by avoiding core file edits.
- Developed with performance in mind for a fast, clutter-free experience.
- Ideal for non-developers looking for a direct and simple way to personalize their email “From” headers.

== Features ==
1. **Simple Configuration** – No coding required; just update the sender name and address in the WordPress admin.
2. **Lightweight** – Minimal footprint ensures no negative impact on site speed.
3. **Compatibility** – Works seamlessly with most WordPress setups and plugins.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/ownomail/` directory, or install via the WordPress "Plugins" screen.
2. Activate the plugin through the "Plugins" screen in WordPress.
3. Navigate to **OwnOmail** in your WordPress Admin menu to configure your sender name and email.

== Frequently Asked Questions ==
**Q:** Will this slow down my site?  
**A:** No. OwnOmail is built to be minimal and efficient, so it won’t negatively impact your website’s performance.

**Q:** Does OwnOmail modify WordPress core files?  
**A:** Absolutely not. All changes are handled through standard WordPress hooks and options.

**Q:** What can I edit with OwnOmail?  
**A:** In keeping with a direct, straightforward approach, OwnOmail only lets you customize the sender name and sender email address—no extra fluff or complexity.

**Q:** Is there a limit to how many times I can edit these settings?  
**A:** No, there is no limit. You can update your sender name or email address anytime via the plugin’s settings page.
